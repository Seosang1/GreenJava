영화예매

1. 비회원 예매 가능

2. 회원은 포인트 지급
 - 배송이 아니므로 주소를 받을 필요 없음

3. 한 번에 한영화의 예매만 가능
 - 한회차에 대해 여러 장의 티켓 예매 가능(좌석 선택 가능)
 - 다른 영화나 다른 회차의 경우 다시 예매해야 함
 - 장바구니 기능 없음
 - 날짜, 회차, 좌석 선택

4. 상품 검색
 - 영화 선택 -> 날짜 및 상영관 검색 -> 회차 검색 -> 예매
 - 날짜 선택 -> 영화 및 상영관 검색 -> 회차 검색 -> 예매
 - 상영관 선택 -> 날짜 및 영화 검색 -> 회차 검색 -> 예매

 - 영화 선택 : 현재 상영중인 영화 목록에서 선택
	검색조건 : 쟝르, 영화명 검색, 관람가능 연령대 검색
 - 날짜 선택 : 달력으로 예매 날짜를 선택(예매 가능 날짜를 표시)
 - 상영관 선택 : 지역과 멀티플렉스내의 상영관을 선택해야 함
	지역 선택 : 도시군 등의 지역을 선택하여 특정 극장을 선택할 수 있어야 함
	멀티플렉스내의 특정 관을 선택할 수 있어야 함

 - 상영관 선택 후 특정 회차 및 좌석 선택(4장 까지 가능)






한 스크린에서 여러 영화를 상영할 수 있음 vs 한 영화는 여러 스크린에서 상영할 수 있음
다 대 다 == n vs n
