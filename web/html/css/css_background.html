배경 지정

각 요소의 배경색을 지정한 경우 background-color 속성을 사용한다. 이 때 요소에 여백을 주면 배경 색상 안쪽으로 여백이 생성된다.

요소에 배경 이미지를 지정할 경우 background-image 속성을 사용한다. 이 때 요소의 범위보다 작은 이미지를 배경이미지로 사용할 경우 이미지가 반복되어 채워지며, background-color 속성이 있어도 이미지에 가려지게 된다.
이미지가 없을 경우 아무런 동작을 하지 않고(에러없이)배경을 그대로 보여줌
