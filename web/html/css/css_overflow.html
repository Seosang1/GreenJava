overflow 속성과 적용결과

overflow는 컨텐츠가 블록 박스의 크기를 넘어가는 경우에 사용하는 속성임. overflow 속성은 visible, hidden, scroll, auto 등의 네 가지 방식이 있고, 기본값은 visible임.
