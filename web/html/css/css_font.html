글자 관련 CSS속성

css에서 글꼴을 지정할 때는 font-family 속성을 이용하여 원하는 글꼴을 지정할 수 있다. 이 때 글꼴명을 여러 개로 지정할 수 있는데 쉼표로 구분하며 입력한 순서가 우선 순위가 된다.

글꼴을 여러 개로 지정하는 이유는 사용자의 시스템에 해당 글꼴이 없을 경우를 대비하기 위해 여러 개를 입력하며, 이 또한 없을 경우를 위해 마지막에 'serif', 'sans-serif', 'cursive', 'fantasy', 'monospace' 중 하나를 지정한다.


css에서의 색상 지정법

글자색을 black, white 등 영문 색상명으로 지정가능

글자색을 6자리의 16진수로 RGB형식을 지정가능

글자색을 총 9자리의 10진수로 RGB형식을 지정가능

글자색을 백분율로 RGB형식을 지정가능
