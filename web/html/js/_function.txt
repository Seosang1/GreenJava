함수(function)
 - 특정 작업을 수행하기 위해 코딩된 프로그램의 한 블록을 의미
 - 함수는 크게 사용자 정의 함수와 내장 함수로 구분됨
 - 함수는 호출을 해야 동작함(자동으로 동작하는 경우는 없음)
 - 함수는 매개변수의 유무에 상관없이 반드시 괄호를 해야 함


1. 사용자 정의 함수
 - 개발자가 직접 만들어서 사용하는 함수
 - 문법
function 함수명([매개변수1, 매개변수2, ... , 매개변수n]) {
	함수내용;
	[return 값;]
}
 - function : 함수를 생성시키는 명령어
 - 함수명 : 변수이름 짓는 방법과 동일한 방법을 사용함
	리턴값이 논리값인 함수는 is로 시작하는 이름을 사용
	논리값이 아닌 리턴값을 가지는 함수는 get으로 시작
	산술적인 계산을 하는 함수는 cal로 시작
 - 매개변수 : 함수가 작업할 때 필요한 데이터를 받아오는 역할로 없을 수도 있음
	매개변수에는 특정 값 뿐만 아니라 요소(태그)같은 객체나 배열도 들어갈 수 있음
	인수, 아규먼트(argument), 파라미터(parameter) 등으로 부르기도 함
	매개변수는 해당 함수내에서만 사용할 수 있는 일종의 지역변수
 - return : 함수의 실행결과를 함수를 호출한 곳으로 리턴하는 명령어

 - 함수는 호출해야 실행되므로 반드시 선언한 함수를 호출하는 코드가 있어야 함
 - 호출하는 것은 사용자가 일으키는 이벤트에 따라 동작됨
 - 함수를 선언하기 전에 호출하면 동작하지 않음 - 단, 자동호출이 아니면 상관없음
	자동호출 : 페이지가 열리면서 자동으로 호출하는 것을 의미


2. 내장 함수
 - 자바스크립트가 자체적으로 제공하는 함수를 의미
 - 시스템(웹 브라우저)에 정의되어 있으므로 따로 정의하지 않아도 사용가능

1) eval()
 - 매개변수로 전달된 문자열을 자바스크립트 명령으로 인식하여 처리하는 함수
 - 문법
eval("문자열"); -> "문자열"을 명령어 바꾸어 처리

2) parseInt(), parseFloat()
 - 숫자형 인수를 정수(parseInt())나 실수(parseFloat())로 변환하는 함수
 - 문법
parseInt("숫자형 문자열" [, 진법]); parseFloat("숫자형 문자열");
 - parseInt()는 반올림이 아닌 내림으로 정수 부분만 취함
 - parseInt()는 진법을 입력하면 인수로 입력된 숫자형 문자열을 2, 8, 16진수로 인식하여 값을 구할 수 있음
 - parseInt()는 문자가 섞이면 숫자부분만 추출하여 정수로 변환

3) isFinite(), isNaN()
 - 인수가 유리수인지 여부와 숫자인지 여부를 논리값으로 리턴하는 함수
 - 문법1 : isFinite(검사할 숫자) : 지정된 숫자가 유리수 이면 true
 - 문법2 : isNaN(검사할 값) : 지정된 값이 숫자이면 false

4) Number(), String()
 - 인수를 각각 숫자와 문자열로 변환하는 함수
 - 문법1 : Number("숫자로 변환할 문자열")
 - 문법2 : String(문자열로 변환할 숫자)
 - Number()에서 숫자로 변환할 문자열에 글자가 섞여 있으면 오류발생(NaN)

5) escape(), unescape()
 - 인수들을 각각 인코딩 및 디코딩해주는 함수
 - 문법1 : escape(인코딩할 문자열)
 - 문법2 : unescape(디코딩할 문자열)
 - 한글 등이 온라인상에서 깨지는 현상을 막기 위해 사용되는 함수들
	주로 url의 쿼리스트링으로 데이터를 주고 받을 때 사용함
