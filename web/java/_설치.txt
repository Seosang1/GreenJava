JDK 설치
 - http://java.sun.com/ 으로 들어가 필요한 JDK를 다운받음 (Java SE : Standard Edition)
	실제 실무에서 설치시엔 반드시 JDK의 버전을 맞춰 설치해야함

 - 예전 버전 설치시 자바를 설치해도 바로 사용할 수 없으므로 윈도우 환경변수를 설정해야함
	환경변수 설정 : [내PC]의 속성 창 -> 좌하단의 [고급시스템 설정] 클릭 -> 시스템 속성 창에서 우하단의 [환경변수] 버튼 클릭 -> 환경변수 창에서 시스템 변수들 중 path변수 더블 클릭 -> 
windows 10 : 환경변수 편집 창에서 우상단의 [새로 만들기] 버튼 클릭후 자바가 설치된 경로에서 bin폴더의 위치를 입력함('C:\Program Files\Java\idk-15.0.2\bin')
windows 7 이하 : path변수의 값들을 세미콜론으로 구분하여 보여주는 창이 뜨고 그 값들의 마지막에 세미콜론 입력 후 자바가 설치된 경로에서 bin폴더의 위치를 입력함('C:\Program Files\Java\idk-15.0.2\bin')


bin폴더안에 있는 javac.exe와 java.exe파일을 어느 곳에서든 실행할 수 있게 하기 위해서 환경변수를 설정함
